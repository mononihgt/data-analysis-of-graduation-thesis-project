# Results Guidelines

## Scope
This directory contains generated analysis outputs.

## Rules
- Write outputs under `results/<analysis_name>/`.
- Prefer regenerating outputs from scripts rather than hand-editing generated files.
- Keep tables, figures, and model summaries together when they belong to the same analysis.
- In handoff notes, state which script produced changed result files.
- When statistical outputs change, keep the accompanying provenance or summary text in the same analysis folder when practical.
- For coordinate-based outputs, keep the reported comparison space in shared 0–10 units; raw pixels and 400×400 values are intermediate or provenance-only representations.
